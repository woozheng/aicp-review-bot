# AICP Review Bot

🤖 AI-powered code review. Reviews every PR automatically.  
🔧 **Not just reviews. It fixes.**

💸 **Saves you $12/month per repo.** Free. Unlimited repos. Forever.

## How it works
Open a PR → Bot reviews it → Posts review with 🔴🟡🟢 severity → **Auto-fixes critical issues** → Posts fix

## Quick Start
1. Download `review-bot.exe` from Releases
2. Set environment variables:
   - `GITHUB_TOKEN` - Your GitHub personal access token
   - `API_KEY` - Your LLM API key
   - `API_URL` - LLM API endpoint
   - `MODEL` - Model name
3. Run `review-bot.exe`
4. Add webhook: `http://your-server:8080/webhook/github`
5. Open a PR → Bot reviews and fixes it

That's it. 5 minutes.
